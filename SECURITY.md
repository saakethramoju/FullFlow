# Security Policy

## Supported Versions

FullFlow is currently in early development. Security fixes will generally target the latest released version.

## Reporting a Vulnerability

Please report security concerns privately by contacting:

skramoju@gmail.com

Do not open a public GitHub issue for security vulnerabilities.

Please include:

- A description of the issue
- Steps to reproduce, if applicable
- Affected versions
- Any known workarounds

I will review the report and respond as soon as practical.
