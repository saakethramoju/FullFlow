# Contributing to FullFlow

Contributions are welcome.

FullFlow is an early-stage engineering simulation framework, so bug reports, examples, documentation improvements, and new components are especially helpful.

## Reporting Issues

When opening an issue, please include:

- A clear description of the problem
- A minimal code example, if possible
- The Python version
- The FullFlow version
- Relevant error messages or solver output
